You will also to create .env file with current variables: PGHOST='host-link'
PORT='port'

SMTP_USER='email'
SMTP_PASSWORD='password'

POSTGRES_HOST=127.0.0.1
POSTGRES_DB='database-name'
POSTGRES_USER='user-name'
POSTGRES_PASSWORD='password'
